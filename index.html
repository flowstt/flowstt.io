Features

Built from the ground up for accurate, private, and responsive voice transcription.

🔒

Privacy-First

All audio processing and transcription happens locally. No data ever leaves your machine. No subscriptions, no signups, no cloud services.

💻

Cross-Platform

Native audio backends for each OS: WASAPI on Windows, PipeWire on Linux, CoreAudio and ScreenCaptureKit on macOS.

⚡

Hardware Accelerated

NVIDIA CUDA on Windows and Linux, Apple Metal on macOS. Falls back to CPU when no GPU is available.

📊

Real-Time Visualization

60fps scrolling waveform and FFT-based spectrogram with a detailed speech activity graph showing detection internals.

🎙

Intelligent Speech Detection

Multi-feature analysis: amplitude, zero-crossing rate, spectral centroid. Dual-mode voiced and whisper detection with transient rejection.

🎤

PTT & Automatic Modes

Push-to-Talk with configurable global hotkeys, or continuous automatic mode with VAD-based segment extraction and word-break timing.

📟

Full CLI

Complete command-line interface with JSON output for headless operation, piping, and scripting. Auto-spawns the background service.

🔊

Echo Cancellation

WebRTC AEC3 algorithm removes speaker feedback when capturing both microphone and system audio simultaneously.

Architecture

Three cooperating components connected via platform-native IPC.

⚙

Service

flowstt-service

Background daemon handling audio capture, speech detection, and transcription. Runs independently of any client.

📟

CLI

flowstt

Command-line interface for headless operation and scripting. Auto-spawns the service if not already running.

🖥

GUI

flowstt-app

Tauri 2.0 desktop application with real-time visualization, configuration window, and system tray integration.

Components communicate over platform-native IPC (Unix sockets on Linux/macOS, named pipes on Windows) using length-prefixed JSON messages. Multiple clients can connect simultaneously; the service broadcasts events to all subscribers.

Getting Started

Get FlowSTT running on your system in minutes.

System Requirements

  • Windows 10+ with WASAPI audio support
  • Linux with PipeWire audio server
  • macOS 12.3+ with CoreAudio / ScreenCaptureKit
  • Optional: NVIDIA GPU (CUDA) or Apple Silicon (Metal) for accelerated inference

Quick Start (CLI)

# List available audio devices
flowstt list

# Start transcribing from your microphone
flowstt transcribe

# Transcribe with echo cancellation (mic + system audio)
flowstt transcribe --input <mic-id> --system <output-id> --echo-cancel

# Check GPU availability
flowstt gpu

# Download the Whisper model
flowstt model download

# View service status
flowstt status

Building from Source

# Clone the repository
git clone https://github.com/keathley/flowstt.git
cd flowstt

# Install frontend dependencies
pnpm install

# Build the desktop application
cargo tauri build

Vision & Roadmap

FlowSTT aims to become a conversational voice agent where timing, cadence, and intent all matter.

🎵

Real-Time Cadence Analysis

Detect natural pauses vs. intentional breaks. Distinguish thinking pauses from end-of-thought pauses. Learn individual speech patterns over time.

⏱

Adaptive Timeouts

Dynamic timeout adjustment based on conversational context. Extended waits for mid-sentence pauses, shorter timeouts for simple commands.

✅

Acknowledgment Feedback

Audible confirmation when input is captured. Processing indicators during transcription. Eliminate the "did it hear me?" uncertainty.

✋

Interrupt Handling

Graceful soft and hard interrupts. Continue speaking during processing or explicitly redirect. Recovery with clarifying questions.

💬

Dynamic Query & Follow-up

Active conversation participation with confidence-driven clarification, contextual questions, and intent assessment before action.

🚀

Workflow Automation

Execute file operations, application control, and system commands from natural language. Chain actions from single conversational exchanges.
